# Lacework Compliance Remediation

Terraform module for remediating common non-compliant resources in AWS as detected by Lacework.

## Requirements

| Name      | Version    |
| --------- | ---------- |
| terraform | >= 0.12.31 |
| aws       | ~> 5.0     |
| lacework  | ~> 1.16     |

## Providers

| Name     | Version |
| -------- | ------- |
| archive  | 2.4.0   |
| aws      | 5.25.0  |
| lacework | 1.17.0  |
| local    | 2.4.0   |
| random   | 3.5.0   |
